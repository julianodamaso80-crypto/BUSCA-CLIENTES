# Glossario de Termos

## A

### Anti-Bloqueio
Sistema de protecao que evita que o numero de WhatsApp seja bloqueado por envio excessivo de mensagens.

### Apify
Plataforma de web scraping que fornece a API para buscar dados do Google Maps.

## B

### Busca
Operacao de pesquisa de clientes no Google Maps por segmento e localizacao.

## C

### Campanha
Conjunto de mensagens a serem enviadas para uma lista de contatos via WhatsApp.

### Chunk
Fragmento semantico de um documento usado para busca por similaridade.

### Contexto
Informacao relevante recuperada do banco de conhecimento para auxiliar respostas.

## D

### Dashboard
Painel principal do sistema com estatisticas e acoes rapidas.

### Delay
Tempo de espera entre o envio de cada mensagem para evitar bloqueio.

### Disparo
Acao de enviar mensagens em massa para uma lista de contatos.

## E

### Embedding
Representacao vetorial de texto usada para busca semantica.

### Evolution API
API REST para integracao com WhatsApp baseada no Baileys.

### Exportacao
Funcionalidade de baixar dados em formato CSV.

## G

### Google Maps
Servico de mapas do Google de onde extraimos informacoes de empresas.

## I

### Instancia
Conexao individual de WhatsApp na Evolution API.

### Ingestao
Processo de adicionar documentos ao sistema de contexto.

## L

### Lista de Bloqueados
Contatos que solicitaram nao receber mais mensagens.

## M

### Mensagem Personalizada
Mensagem com variaveis que sao substituidas pelos dados do cliente.

## P

### Pausa Periodica
Intervalo maior entre grupos de mensagens para simular comportamento humano.

## Q

### QR Code
Codigo que o usuario escaneia para conectar o WhatsApp.

### Query
Consulta de busca no sistema de contexto.

## R

### Rastreabilidade
Capacidade de identificar a origem de cada resposta no sistema.

## S

### SaaS
Software as a Service - modelo de negocio do BUSCA CLIENTES.

### Segmento
Tipo de negocio buscado (ex: restaurantes, oficinas).

### Sincronizacao
Processo de atualizar o banco de conhecimento com novos documentos.

## T

### Token
Unidade de texto usada para medir tamanho de contexto.

## V

### Vetor
Representacao numerica de texto para busca por similaridade.

### Versao
Snapshot do estado do sistema de contexto em um momento.

## W

### Webhook
Notificacao automatica enviada pela API quando ocorre um evento.

### WhatsApp
Aplicativo de mensagens usado para enviar campanhas.
